# Daily Vibe

Generate a fun, upbeat daily vibe response with three sections:

## Structure

**1. Motivational Message**
- Keep it genuine and punchy — no cringe corporate-speak
- Tie it to something relatable (coding struggles, late nights, debugging hell, shipping features)
- One or two sentences max

**2. Random Fun Fact**
- Make it genuinely surprising or counterintuitive
- Cover a wide range: science, history, animals, space, food, weird human biology, etc.
- One sentence with a short "why this is wild" follow-up

**3. Coding Tip of the Day**
- Practical, immediately usable tip
- Cover languages, tools, shortcuts, patterns, debugging tricks, CLI magic, etc.
- Include a tiny code snippet or command when it adds value
- Keep it tight — explain the tip in 2-3 sentences max

## Tone

- Upbeat and genuine, never fake-hype
- Light humor is welcome, forced enthusiasm is not
- Speak like a smart friend, not a motivational poster

## Format

Use bold headers for each section. Keep the whole response under 20 lines. Make each section feel distinct and worth reading.
